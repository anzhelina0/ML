# Домашнее задание 2

## Создание и оптимизация ResNet18 для Tiny ImageNet

### Цель работы

Поэтапно реализовать кастомную архитектуру **ResNet18** для классификации изображений из датасета **Tiny ImageNet**, выбрать 10 классов и исследовать влияние архитектурных решений на качество модели.

---

## Используемый датасет

**Tiny ImageNet** (200 классов, изображения 64×64).

В работе используются **10 выбранных классов**:

```
n01443537, n01629819, n01641577, n01644900, n01698640,
n03089624, n01770393, n01774750, n03976657, n02395406
```

Датасет автоматически загружается с помощью `kagglehub`.

## Архитектура модели

Реализована кастомная ResNet18:

* BasicBlock: `Conv3x3 → BN → ReLU → Conv3x3 → BN → Skip → ReLU`
* Конфигурация каналов: `64 → 128 → 256`
* Количество блоков: `[2, 2, 2]`
* AdaptiveAvgPool + Linear слой

Общее число параметров: **менее 5M** (ограничение соблюдено).

---

## Обучение

* Оптимизатор: **Adam**
* Learning rate: `0.001`
* Loss: **CrossEntropyLoss**
* Эпохи: `20`

Логируются метрики:

* Train Accuracy / Loss
* Validation Accuracy / Loss
